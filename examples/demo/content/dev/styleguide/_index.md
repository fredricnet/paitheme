# PAI Theme Styleguide

This styleguide demonstrates all available components, shortcodes, and design patterns in the PAI Theme. All examples use explanatory text that describes what each section demonstrates.

## Purpose

This styleguide serves as both:

1. **Development Tool** - Test theme changes and see how components render
2. **Theme Documentation** - Showcase theme features with self-documenting examples
3. **Reference Guide** - Quick reference for available components and their usage
