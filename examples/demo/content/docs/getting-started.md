# Getting Started with PAI Theme

This page demonstrates how documentation pages look in PAI Theme.

## Installation

Install PAI Theme as a Hugo module:

```bash
hugo mod init github.com/your-org/your-site
hugo mod get github.com/fredricnet/paitheme
```

## Configuration

Add the theme to your `hugo.yaml`:

```yaml
module:
  imports:
    - path: github.com/fredricnet/paitheme
```

## Content Structure

This page shows how documentation content is formatted:

- **Headings** create the table of contents automatically
- **Code blocks** are highlighted with syntax highlighting
- **Lists** like this one are styled cleanly
- **Links** are clearly visible and accessible
