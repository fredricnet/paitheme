# Blog

This section demonstrates the **blog layout** features of PAI Theme, including:

- Clean blog post listings
- Featured images
- Tags and categories
- Author information
- Publication dates
- Post summaries

## Blog Features

PAI Theme's blog section includes:

1. **List View** - Clean, organized list of blog posts
2. **Featured Images** - Support for post header images
3. **Tags** - Easy tagging and filtering
4. **Metadata** - Author, date, and reading time
5. **Excerpts** - Automatic or manual post summaries
