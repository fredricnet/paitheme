# Theme

A **theme** is a collection of templates and assets that define the appearance and functionality of a Hugo site. Themes can include layouts, shortcodes, static files, and configuration templates.

## Theme Components

- **Layouts**: HTML templates for different content types
- **Shortcodes**: Reusable content snippets
- **Static Files**: CSS, JavaScript, images
- **Configuration**: Default settings and parameters

## PAI Theme

PAI Theme is a modern Hugo theme designed for documentation sites, blogs, and marketing pages with a focus on dark mode design and clean aesthetics.
