# Image Assets Required

This theme requires the following image assets to be placed in the `static/images/` directory.

## Required Images

### Hero Section
- `static/images/hero/hero-image.webp` - Main hero image showing the product interface with connected banks
- `static/images/hero/dashboard-foreground.webp` - Foreground version for the value proposition section

### Features Section
- `static/images/features/balance-overview.webp` - Balance Overview feature screenshot
- `static/images/features/transaction-ledger.webp` - Transaction Ledger feature screenshot
- `static/images/features/initiate-transfers.webp` - Initiate Transfers feature screenshot
- `static/images/features/generate-export.webp` - Generate & Export feature screenshot

### Logo Images (SVG format preferred)
- `static/images/logos/mercury.svg` - Mercury logo
- `static/images/logos/watershed.svg` - Watershed logo
- `static/images/logos/retool.svg` - Retool logo
- `static/images/logos/descript.svg` - Descript logo
- `static/images/logos/perplexity.svg` - Perplexity logo
- `static/images/logos/monzo.svg` - Monzo logo
- `static/images/logos/ramp.svg` - Ramp logo
- `static/images/logos/raycast.svg` - Raycast logo
- `static/images/logos/arc.svg` - Arc logo

## How to Download

1. Right-click on each image and save:
   - Hero dashboard image
   - Feature tab images (switch between tabs to see different images)
2. Save images in WebP format for optimal performance
3. Place them in the corresponding directories listed above

## Image Specifications

- Format: WebP (preferred) or PNG
- Hero images: Recommended width 1200-1600px
- Feature images: Recommended width 800-1200px
- Optimize images for web (compress while maintaining quality)

## Fallback

If images are not available, the theme will display placeholder divs with descriptive text. However, for the best experience, use the actual images.
