# Exploring PAI Theme Features

This is an example blog post that demonstrates how blog content looks in PAI Theme.

## Blog Post Structure

Blog posts in PAI Theme support:

- **Featured images** (if specified in frontmatter)
- **Tags** for categorization
- **Author information**
- **Publication dates**
- **Markdown content** with full formatting support

## Content Formatting

You can use all standard Markdown features:

- **Bold text**
- *Italic text*
- [Links](https://example.com)
- Lists (ordered and unordered)
- Code blocks
- And more!
