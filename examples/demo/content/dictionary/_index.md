# Dictionary / Glossary

This section demonstrates the **dictionary/glossary** functionality of PAI Theme. Dictionary entries are:

- Alphabetically organized
- Expandable for detailed definitions
- Searchable
- Perfect for technical terms, glossaries, and definitions

## How It Works

The dictionary section automatically:

1. Organizes entries alphabetically
2. Creates a clean list view
3. Provides expandable entries for detailed definitions
4. Supports structured data (JSON-LD) for AI systems
