# Welcome to PAI Theme

This is a **demo site** that showcases all the features and capabilities of the PAI Theme. Browse through the different sections to see how the theme works and what it can do.

## What is PAI Theme?

PAI Theme is a modern, dark-mode Hugo theme designed for documentation sites, blogs, and marketing pages. It features:

- **Dark Mode First** - Elegant dark theme with clean design
- **Documentation System** - Full-featured docs with sidebar navigation
- **Dictionary/Glossary** - Alphabetically organized glossary with expandable entries
- **Blog Support** - Clean blog layout with featured images
- **SEO Ready** - JSON-LD structured data for search engines
- **AI Ready** - Auto-generated llms.txt and robots.txt
